Cyclistic Bike-Share Analysis

Google Data Analytics Capstone Project

This project analyzed bike-share usage data to uncover patterns between casual and annual riders, aiming to provide strategic recommendations for increasing annual memberships. I followed the six-step data analysis process: Ask, Prepare, Process, Analyze, Share, and Act.

Key Contributions

  • Cleaned and merged multiple large CSV datasets using Excel and R.
  • Performed data analysis on ride duration, user type, and weekday usage patterns.
  • Created visualizations that illustrated key behavioral trends between user groups.
  • Presented recommendations on marketing strategies and membership conversion.

Skills & Tools Applied

Data Cleaning • Data Visualization • Business Problem Framing • Excel • R • Data Storytelling
